###华为认证
---
1.华为认证是华为技术有限公司打造的覆盖ICT全技术领域的认证体系。

2.华为认证可分为三个等级：HCIA、HCIP、HCIE。自2019年3月起，考HCIP各个方向都不需要HCIA证书。华为考试科目及费用等，这里不再列举，可自行网上查询。
